Snowflake

Snowflake is a Parsing Expression Grammar (PEG) library and graphical parser-generator/editor.

View project onGitHub

This site is still under construction.

Snowflake is actually two projects in one.

Part I: The Library

The library facilitates the runtime creation of Parsing Expression Grammar recursive-descent parsers.

Quick Links:

  • Download the library as a precompiled .jar file.
  • View the library Javadoc documentation.
  • Browse the source code.
  • Download the library source code as a .zip file.

Part II: The Interactive Graphical Grammar Editor

The grammar editor provides users the ability to instantly test their grammars on user-specified input.

Watch a tutorial now!

Steps:

  1. Start the grammar editor. Try it Now Online!

  2. Enter a custom PEG grammar into the "Grammar" tab.

  3. Enter custom input, to be parsed, into the "Input" tab.

  4. Click Action → Parse.

  5. The parse-tree created by parsing the input using the grammar will now be displayed.

  6. Repeat the previous steps, until you have completed your grammar.
