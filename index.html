Snowflake

Snowflake lets you dynamically create recursive-descent parsers.

Download this project as a .zip file Download this project as a tar.gz file

This site is still under construction.

Snowflake is actually two projects in one, each of which are described below.

Snowflake is licensed under the Apache 2.0 licence.

Part I: The Library

The library facilitates the creation of PEG-based recursive-descent parsers at runtime.

Quick Links:

  • Download the library as a precompiled .jar file.
  • Download the library source code as a .zip file.
  • Browse the latest source code.

Part II: The Interactive Graphical Grammar Editor

The grammar editor provides the ability to instantly test grammars on user-specified input.

Quick Links:

  • View a getting-start slideshow.
  • Cheat Sheet
  • Download the grammar editor as a single precompiled .jar file.
  • Download the grammar editor's source code as a .zip file.

Part III: What can you use Snowflake for?

Snowflake allows you to create PEG-based recursive-descent parsers for use in J program via an easy to use graphical interface.

Downloadable Examples:

  • A Simple Boolean Expression Evaluator
  • A Simple Phone Book Parser
  • A Prefix Notation Calculator
  • An Infix Notation Calculator
